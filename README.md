FAQ Chatbot

Description

An AI-powered FAQ Chatbot built using Python and Scikit-learn. The chatbot uses Natural Language Processing (NLP), TF-IDF Vectorization, and Cosine Similarity to understand user queries and provide relevant answers from a predefined FAQ database.

Features

- Answers frequently asked questions automatically
- Uses NLP for intelligent question matching
- Provides quick and accurate responses
- Easy to customize with new FAQs
- User-friendly command-line interface

How to Run

1. Install the required library:
   pip install scikit-learn

2. Run the chatbot:
   python faq_chatbot.py

3. Enter your question and get a response.

Project Objective

The objective of this project is to develop an intelligent FAQ chatbot that can understand user queries and provide accurate responses using NLP techniques.
